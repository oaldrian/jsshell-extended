# minesweeper

Classic minesweeper with a cursor and flags.

## Usage

- `minesweeper`

## Goal

- Reveal all safe tiles without stepping on a mine.
- Use numbers to deduce where mines are.

## Controls

- Arrow keys: move cursor
- `Space` / `Enter`: reveal
- `f`: flag/unflag
- `r`: restart
- `Esc` / `q`: quit

## Notes

- You can pass custom size/mines: `minesweeper <w> <h> <mines>`

## Tips

- If the screen gets messy, run `cls`.
