# 2048

Slide and merge tiles to reach higher numbers.

## Usage

- `2048`

## Goal

- Combine equal tiles (2+2, 4+4, ...) to build bigger numbers.
- The game ends when no moves remain.

## Controls

- Arrow keys: move tiles
- `r`: restart
- `Esc` / `q`: quit

## Tips

- Try to keep the biggest tile in a corner.
- Avoid filling the board with scattered small tiles.
- If the screen gets messy, run `cls`.
