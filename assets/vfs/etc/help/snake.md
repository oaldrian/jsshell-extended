# snake

Classic snake: eat food, grow longer, don’t crash.

## Usage

- `snake`

## Goal

- Eat food (`●`) to grow and increase your score.
- Avoid walls and your own body.

## Controls

- Arrow keys: move
- `p`: pause/resume
- `r`: restart
- `Esc` / `q`: quit

## Notes

- You can pass a speed in ms: `snake 120` (lower is faster).

## Tips

- If the screen gets messy, run `cls`.
