# tetris

Falling-block puzzle game.

## Usage

- `tetris`

## Goal

- Clear lines by filling rows completely.
- Survive as the pieces fall faster each level.

## Controls

- Left/Right: move
- Down: soft drop
- Up: rotate
- Space: hard drop
- `p`: pause
- `r`: restart
- `Esc` / `q`: quit

## Tips

- If the screen gets messy, run `cls`.
