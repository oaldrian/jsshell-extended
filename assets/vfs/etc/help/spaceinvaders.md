# spaceinvaders

Arcade shooter: clear waves of invaders.

## Usage

- `spaceinvaders`

## Goal

- Shoot all invaders before they reach the bottom.
- Survive with your limited lives.

## Controls

- Left/Right: move
- Space/Enter: shoot
- `p`: pause
- `r`: restart
- `Esc` / `q`: quit

## Notes

- Clearing a wave advances to the next level (faster + harder).

## Tips

- If the screen gets messy, run `cls`.
