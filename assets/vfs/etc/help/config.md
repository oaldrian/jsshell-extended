# config

View or change shell appearance/settings.

## Usage

- `config` (show)
- `config get <key>`
- `config set <key> <value>`
- `config reset`

## Examples

- `config`
- `config get theme`
- `config set theme dark`

## Notes

- Settings are persisted in the VFS (see `/sys/config.json`).
